## 4. Update Vite Configuration

### File: `vite.config.js`
Add path alias for components directory:

```javascript
import { fileURLToPath, URL } from 'node:url'

import { defineConfig } from 'vite'
import vue from '@vitejs/plugin-vue'
import vueDevTools from 'vite-plugin-vue-devtools'

// https://vite.dev/config/
export default defineConfig({
  plugins: [
    vue(),
    vueDevTools(),
  ],
  resolve: {
    alias: {
      '@': fileURLToPath(new URL('./src', import.meta.url)),
      '@com': fileURLToPath(new URL('./src/components', import.meta.url)),
    },
  },
  server: {
    host: '0.0.0.0',
    port: process.env.VITE_HMR_PORT,
    hmr: {
      protocol: 'ws',
      port: process.env.VITE_HMR_PORT,
    },
    watch: {
      usePolling: true,
      useFsEvents: true,
      interval: 1000,
    }
  }
})
```

## 8. Configure Vue Router

### File: `src/router/index.js`
Set up routing with authentication pages:

```javascript
import Signin from '@com/auth/Signin.vue';
import Signout from '@com/auth/Signout.vue';
import Signup from '@com/auth/Signup.vue';
import VerifyEmail from '@com/auth/VerifyEmail.vue';
import ResetPassword from '@com/auth/ResetPassword.vue';
import SetNewPassword from '@com/auth/SetNewPassword.vue';
import Dashboard from '@com/pages/Dashboard.vue';

import { createRouter, createWebHistory } from 'vue-router'

const router = createRouter({
  history: createWebHistory(import.meta.env.BASE_URL),
  routes: [
    {
      path: '/',
      name: 'auth.signin',
      component: Signin,
    },
    {
      path: '/signout',
      name: 'auth.signout',
      component: Signout,
    },
    {
      path: '/signup',
      name: 'auth.signup',
      component: Signup,
    },
    {
      path: '/email/verify/:api_url',
      name: 'auth.verify.email',
      component: VerifyEmail,
    },
    {
      path: '/password/reset',
      name: 'auth.reset.password',
      component: ResetPassword,
    },
    {
      path: '/password/reset/:api_url',
      name: 'auth.set.password',
      component: SetNewPassword,
    },
    {
      path: '/dashboard',
      name: 'dashboard',
      component: Dashboard,
    }
  ],
})

export default router
```

### Route Explanations:
- **`/`**: Main login page (Signin component)
- **`/signup`**: User registration page
- **`/signout`**: Logout functionality
- **`/email/verify/:api_url`**: Email verification page with dynamic API URL parameter
- **`/password/reset`**: Password reset request page
- **`/password/reset/:api_url`**: New password setting page with API URL parameter
- **`/dashboard`**: Protected dashboard page for authenticated users

## 11. Understanding the Router Structure

### Route Parameters Explanation:

#### Email Verification Route:
```javascript
{
  path: '/email/verify/:api_url',
  name: 'auth.verify.email',
  component: VerifyEmail,
}
```

- **`:api_url`**: Dynamic parameter containing the encoded backend API URL
- **Example URL**: `/email/verify/http%3A//localhost%3A8000/api/email/verify/1/abc123...`
- **Purpose**: Receives the verification URL from Laravel backend email

#### Password Reset Route:
```javascript
{
  path: '/password/reset/:api_url',
  name: 'auth.set.password',
  component: SetNewPassword,
}
```

- **`:api_url`**: Contains the password reset API URL with token
- **Example URL**: `/password/reset/http%3A//localhost%3A8000/api/password/reset?token=xyz...`
- **Purpose**: Handles password reset flow from email links

## 13. Integration with Laravel Backend

### API Integration Points:
1. **Signin form** → POST `/api/signin`
2. **Signup form** → POST `/api/signup`  
3. **Password reset** → POST `/api/password/forgot`
4. **New password** → POST `/api/password/reset`
5. **Email verification** → GET extracted from `:api_url` parameter

### URL Parameter Handling:
The `:api_url` parameters in routes will contain URL-encoded API endpoints from Laravel:
- Email verification: Laravel sends user to frontend with encoded verification URL
- Password reset: Laravel sends user to frontend with encoded reset URL
- Frontend extracts and calls these URLs to complete the process
